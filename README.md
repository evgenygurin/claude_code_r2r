   1 # R2R + Claude Code Integration
   2 
   3 Полная интеграция R2R (Retrieval-Augmented Generation платформа) с Claude Code через MCP, Plugins, Subagents, Skills, и Hooks.
   4 
   5 ## Особенности
   6 
   7 - ✅ **Нулевое ожидание**: асинхронная загрузка данных через Hatchet workflows
   8 - ✅ **Параллельность**: фоновые задачи в Claude Code + background bash commands
   9 - ✅ **Streaming**: RAG/Agent ответы в реальном времени
  10 - ✅ **Автоматизация**: hooks для автоингестинга при изменении файлов
  11 - ✅ **Переиспользование**: Skills для автоподключения R2R в любом проекте
  12

  23 ## Задача
  24 
  25 Спроектировать и реализовать интеграцию R2R API с Claude Code, используя:
  26 
  27 1. **MCP (Model Context Protocol)** - основной мост к R2R API
  28 2. **Plugins** - упаковка всего в переиспользуемый компонент
  29 3. **Subagents** - специализированные агенты для разных R2R-задач
  30 4. **Agent Skills** - автоматическое подключение R2R функционала
  31 5. **Hooks** - реакция на события (file changes, session start, etc.)
  32 6. **Headless mode** - автоматизация через CI/CD, cron, scripts
  33 7. **Output styles** - опционально, для специфичных форматов

  48 ## Критерии успеха
  49 
  50 1. Пользователь спрашивает "Найди информацию о X" → Claude автоматически использует R2R
  51 2. Пользователь загружает папку → загрузка немедленная, работа продолжается
  52 3. Пользователь редактирует файл → автоматическая синхронизация с R2R в фоне
  53 4. CI/CD автоматически индексирует документацию при deploy
  54 5. Всё работает прозрачно для пользователя
